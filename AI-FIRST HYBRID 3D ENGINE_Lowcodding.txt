    PROYECTO
AI-FIRST HYBRID 3D ENGINE

Objetivo

Un motor 3D que funcione de tres maneras:

1️⃣ Modo Manual
usuario crea todo sin IA (motor tradicional)

2️⃣ Modo API AI
usa modelos externos como OpenAI para generar código, assets o lógica

3️⃣ Modo Local AI
usa un modelo open-source local (MLL) como cerebro del editor

El usuario puede alternar con un toggle switch.
